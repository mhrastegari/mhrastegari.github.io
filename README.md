# MH Rastegari Portfolio

Personal portfolio and resume website for Mohammad Hossein Rastegarinia / MH Rastegari.

## Scripts

- `npm run dev` starts the local Vite dev server.
- `npm run build` type-checks and builds the GitHub Pages-ready static output.
- `npm run lint` runs ESLint.
- `npm run preview` previews the production build locally.
